RMS Titanic · StackingClassifier

Stacking Ensemble

Five diverse base learners combined via a meta-learner

Architecture

Unlike soft-voting which simply averages probabilities, stacking trains a meta-learner (Logistic Regression) on out-of-fold predictions from five diverse base estimators. Each base model captures different decision boundaries — linear, tree-based, kernel-based, and distance-based — so the meta-learner learns how to weight them optimally. No feature engineering or hyperparameter tuning is applied.
